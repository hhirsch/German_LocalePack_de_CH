Deutsches Magento Sprachpaket für die Schweiz

(Magento Community Modules - German (Switzerland) Language Pack)

(Paquete de idioma alemán para Magento Commerce)

Wichtiger Hinweis zur Magento Version 1.9

Magento Inc. hat mit der 1.9 ein eigenes, teilautomatisch übersetztes, Sprachpaket vorinstalliert. Wer das LocalePack_de_CH benutzen möchte, muss vorher den Ordner app/locale/de_CH löschen, damit dieses Sprachpaket installiert werden kann.

Nota importante acerca Magento Versión 1.9

Magento Inc. tiene con la versión 1.9 su propio, semi-automáticamente traducido, paquete de idiomas como preinstalado. El que quiere usar el LocalePack_de_CH, primero debe eliminar la carpeta app/locale/de_CH para que el paquete de idioma se puede instalar.
